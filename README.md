# Freelance Blog Service

開発ログ・進捗共有・技術記事を一元管理するためのブログサービスです。

SES業務で納期管理や進捗共有に苦労した経験から、「途中状態を共有できる仕組み」の重要性を感じ、このサービスの開発を始めました。

単なるブログではなく、

* 開発ログ
* 作業時間記録
* タスク進捗
* 技術アウトプット

を継続的に可視化し、エンジニアが安心して進捗共有できる環境を目指しています。

将来的には、自分自身だけでなく、同じように悩むフリーランスエンジニアや開発者の支援につながるサービスへ成長させたいと考えています。

## Tech Stack

* Python
* FastAPI
* SQLAlchemy
* SQLite
* Git / GitHub
* Docker（予定）
* PostgreSQL（予定）
